python内置函数分析(1)

PYTHON中内置了一些巧妙而强大的内置函数，熟练使用这些函数，代码不仅简洁易读了许多，既方便有减少了很多的BUG。

##1.sorted()##

###1)对一个列表进行排序###
a = [100,82,12,55,45]
sorted(a)

>>>[12,45,55,88,100]

###2)通过key参数/函数###
例：长列表中嵌套字典元素,按照每个元素的长度进行排序(这里使用匿名函数lambda)

list1 = [{2:6,8:2},{1:7},{1:2,5:12,9:2}]
new_list = sorted(list1,key = lambda x:len(x))
print(new_list)
>>>[{1: 7}, {8: 2, 2: 6}, {1: 2, 5: 12, 9: 2}]

###3)对由tuple组成的list进行排序###
例：对一群人里面的年龄的list进行排序

people = [('laownag','man','23'),('xiaoli','woman','21'),('xiaozhang','man','17')]
new_people = sorted(people,key = lambda x:x[2])
print(new_people)
>>>[('xiaozhang', 'man', '17'), ('xiaoli', 'woman', '21'), ('laownag', 'man', '23')]

###4)用cmp函数排序###
python3中，cmp函数被operator模块取代，需要导入
cmp(x,y),如果x>y,返回1，x<y,返回-1，x==y,返回0

people = [('laownag','man','23'),('xiaoli','woman','21'),('xiaozhang','man','17')]
new_people = sorted(people,cmp = lambda x，y:cmp(x[0],y[0])))
print(new_people)
>>>[('laownag', 'man', '23'), ('xiaoli', 'woman', '21'), ('xiaozhang', 'man', '17')]

##2.map()
map函数跟根据提供的函数对制定序列做排序，接受一个函数f和一个list，并把函数f以此作用于每个list函数，然后返回一个新的list,map的函数的参数可以为多个，注意:这个函数一定要有返回值

def func(x):
    return x*x
print(map(func([1,2,3,4,5])))
>>>[1, 4, 9, 16, 25]




def add_x_y(x,y):
return x+y
print (map(add_x_y,[1,2,3,4,5],[2,3,4,5,6]))#必须len(x)==len(y)
>>>[3, 5, 7, 9, 11]

##3.enumerate()
enumerate函数用于遍历序列中的元素以及它们的下标,多用于在for循环中得到计数,enumerate参数为可遍历的变量，如 字符串，列表等

#enumerate函数定义

    'Generates an indexed series: (0,coll[0]), (1,coll[1]) ...'

##4.zip()##

zip函数接收人一多个(包括0或1)序列作为参数，返回一个tuple

#示例1
x=[1,2,3]
y=[4,5,6]
z=[7,8,9]
xyz=zip(x,y,z)
print(xyz)
>>>[(1, 4, 7), (2, 5, 8), (3, 6, 9)]

>

#示例2
x = [1, 2, 3]
y = [4, 5, 6, 7]
xy = zip(x, y)
print xy
>>>[(1, 4), (2, 5), (3, 6)]

>

#示例3
x = [1, 2, 3]
y = [4, 5, 6]
z = [7, 8, 9]
xyz = zip(x, y, z)
u = zip(*xyz)
print u
>>>[(1, 2, 3), (4, 5, 6), (7, 8, 9)]

>
一般认为这是一个unzip的过程，它的运行机制是这样的：
在运行zip(*xyz)之前，xyz的值是：[(1, 4, 7), (2, 5, 8), (3, 6, 9)]
那么，zip(*xyz) 等价于 zip((1, 4, 7), (2, 5, 8), (3, 6, 9))
所以，运行结果是：[(1, 2, 3), (4, 5, 6), (7, 8, 9)]

#示例4
x = [1, 2, 3]
r = zip(* [x] * 3)
print r
>>>[(1, 1, 1), (2, 2, 2), (3, 3, 3)]

>
它的运行机制是这样的：
[x]生成一个列表的列表，它只有一个元素x
[x] * 3生成一个列表的列表，它有3个元素，[x, x, x]
zip(* [x] * 3)的意思就明确了，zip(x, x, x)

##5.filter()##

filter函数是接收一个字符串f,和一个list，这个函数是对每个函数进行判断返回True或者False,依次来过滤掉一些不符合条件的元素，返回符合元素的list.跟if判断添加list相似

def is_num(x):
    return x%2==0
print(filter(is_num,[1,2,3,4,5,6,7]))
>>>[2, 4, 6]

处理文件时，把一些不需要的空格，回车，空字符去掉

def is_not_empty(x):
'strip() 方法用于移除字符串头尾指定的字符（默认为空格）'
    return x in len(x.strip())>0#不返回空列表
print(filter(is_not_empty,['abc','','book','apple','\t','\n']))
>>> ['abc', 'book', 'apple']
