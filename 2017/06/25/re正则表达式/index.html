正则表达式总结

一、检验数字的表达式

  1. 数字: ^[0-9]*$

    #也可以写作 m = re.match("^\d*$","32873")
m = re.match("^[0-9]*$","32873")
print m.group()
>>>32873

  2. n位的数字:^[\d]{n}$

    m = re.match("^\d{4}$","4565")
print m.group()
>>>4565

  3. 至少n位数字:^[\d]{n,}$

    m = re.match("^\d{4,}$","453265")
print m.group()
>>>453265

  4. m-n位的数字:^[\d]{m,n}$

    m = re.match("^\d{4,12}$","453266")
print m.group()
>>>453266

  5. 零和非零开头的数字:^(0|[1-9][0-9]*)$ 

    #a=0,可以 a=0124 不可以
m = re.match("^(0|[1-9][0-9]*)$","787")
print m.group()

  6. 零和非零开头的数字，最多带两位小数数字：^(0|[1-9][0-9]*)+(.[0-9]{1,2})?$

    m = re.match("^(0|[1-9][0-9]*)+(.[0-9]{1,2})?$","7850.04")
print m.group()
>>>7850.04

  7. 带一到两位小数的整数或负数 :^(-)?\d+(.\d{1,2})?$

    m = re.match("^(\-)?\d+(\.\d{1,2})?$","-7850.00")
print m.group()
>>>-7850.00

  8. 整数，负数，小数: ^(-|+)?\d+(.\d+)?$

    #有BUG，0既不是整数，也不是负数，单也可以匹配
m = re.match("^(\-|\+)?\d+(\.\d+)?$","+0.2")
print m.group()
>>>+0.2

  9. 有两位小数的正实数:^(\d+)+(.\d{2})$

    m = re.match("^(\d+)+(\.\d{2})$","12.00")
print m.group()
>>>12.00

10.有一到三位的正实数：^(\d+)+(.\d{1,3})$ 

m = re.match("^(\d+)+(\.\d{1,3})$","12.000")
print m.group()

  1. 非零的正整数: ^(+)?[1-9]\d*$

    m = re.match("^(\+)?[1-9]\d*$","+12")
print m.group()
>>>+12

12.非零的负整数:^(+)?[1-9]\d*$

m = re.match("^(\-)[1-9]\d*$","-12")
print m.group()
>>>-12

  1. 非负整数： ^(+)?\d+$

    m = re.match("^(\+)?\d+$","03")
print m.group()
>>>03

  2. 非正整数:^(-)\d+$

    m = re.match("^(\-)\d+$","-3")
print m.group()
>>>-3
