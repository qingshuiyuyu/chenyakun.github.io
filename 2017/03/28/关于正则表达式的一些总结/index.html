关于正则表达式的一些总结

Posted on 2017-03-28

正则表达式

正则表达式：(Regular Expression)是计算机科学的一个概念。Regular Expression即“描述某种规则的表达式”之意。

正则表达式使用单个字符串来描述、匹配一系列匹配某个句法规则的字符串。在很多文本编辑器里，正则表达式通常被用来检索、替换那些匹配某个模式的文本。

1. 在Python中需要通过正则表达式对字符串进行匹配的时候通常引入：re模块

re.match是用来进行正则匹配检查的方法，若字符串匹配正则表达式，则match方法返回匹配对象（Match Object），否则返回None（注意不是空字符串””）。

匹配对象Macth Object具有group方法，用来返回字符串的匹配部分。

2. re模块示例(匹配以itcast开头的语句)

#coding=utf-8

import re

result = re.match("itcast","itcast.cn")

result.group()

运行结果为：

itcast

3. re.match() 能够匹配出以xxx开头的字符串。

4. 正则表达式中的表示字符

字符|代号

|||

.| 匹配任意1个字符（除了\n）

[ ] |匹配[ ]中列举的字符

\d |匹配数字，即0-9

\D | 匹配非数字，即不是数字

\s | 匹配空白，即 空格，tab键

\S | 匹配非空白

\w | 匹配单词字符，即a-z、A-Z、0-9、_

\W | 匹配非单词字符

需求：匹配出，一个字符串第一个字母为大小字符，后面都是小写字母并且这些小写字母可有可无

示例一：.

<<<re,match(".","m")
"m"
<<<

示例二·：[]

<<<re.match("[Hh]","hello world")#无论h大小写均可
"h"
<<<

5.Python中字符串前面加上 r 表示原生字符串

>>> ret = re.match(r"c:\\a",mm).group()
>>> print(ret)
c:\a

6.正则表达式中表示数量

字符| 功能

*|匹配前一个字符出现0次或者无限次，即可有可无

+|匹配前一个字符出现1次或者无限次，即至少有1次

?|匹配前一个字符出现1次或者0次，即要么有1次，要么没有

{m} |匹配前一个字符出现m次

{m,} |匹配前一个字符至少出现m次

{m,n} |匹配前一个字符出现从m到n次

示例一：*

>>>#coding =utf-8
>>>ret = re.match("[A-Z][a-z]*","Mm")
>>>ret.group()
"Mm"
>>>

>>>ret = re.match("[A-Z][a-z]*","Aabcdef")
>>>ret.group()
"Aabcdef"
>>>

示例二:?

需求：匹配出，0到99之间的数字

#coding=utf-8
import re

ret = re.match("[1-9]?[0-9]","7")
ret.group()

ret = re.match("[1-9]?[0-9]","33")
ret.group()

ret = re.match("[1-9]?[0-9]","09")
ret.group()

7.正则表达式表示边界

字符| 功能

^ |匹配字符串开头

$ |匹配字符串结尾

\b |匹配一个单词的边界

\B |匹配非单词边界

示例一：$

需求：匹配163.com的邮箱地址

#coding=utf-8

import re
ret = re.match("[\w]{4,20}@163\.com$", "xiaoWang@163.comheihei")
ret.group()

结果：报错！！！

8.正则表达式匹配分组

字符 |功能
|—-|——-|

| |匹配左右任意一个表达式

(ab) |将括号中字符作为一个分组

\num |引用分组num匹配到的字符串

(?P) |分组起别名

(?P=name) |引用别名为name分组匹配到的字符串

示例二:()

>>> ret = re.match("([^-]*)-(\d+)","010-12345678")
>>> ret.group()
'010-12345678'
>>> ret.group(1)
'010'
>>> ret.group(2)
'12345678'

9.re模块的高级用法

serch

需求：匹配出文章阅读的次数

#coding=utf-8
import re

ret = re.search(r"\d+", "阅读次数为 9999")
ret.group()

运行结果：

>>>ret.group()
"9999"
>>>

findall

需求：统计出python、c、c++相应文章阅读的次数

#coding=utf-8
import re

ret = re.findall(r"\d+", "python = 9999, c = 7890, c++ = 12345")
print ret

运行结果：

>>>print ret
["9999","7890","12345"]
>>>

sub :将匹配到的数据进行替换

需求：将匹配到的阅读次数加1

方法一：

#coding=utf-8
import re

ret = re.sub(r"\d+", '998', "python = 997")
print ret

运行结果:

>>>print ret
python = 998
>>>

方法二：

#coding=utf-8
import re

def add(temp):
    strNum = temp.group()
    num = int(strNum) + 1
    return str(num)

ret = re.sub(r"\d+", add, "python = 997")
print ret

ret = re.sub(r"\d+", add, "python = 99")
print ret

运行结果:

>>>ret = re.sub(r"\d",add,"python = p77")
>>>print ret
python = 998
>>>ret = re.sub(r"\d",add,"python = 99")
print ret
>>>python = 100

10.贪婪和非贪婪

Python里数量词默认是贪婪的（在少数语言里也可能是默认非贪婪），总是尝试匹配尽可能多的字符；

非贪婪则相反，总是尝试匹配尽可能少的字符。

在”*”,”?”,”+”,”{m,n}”后面加上？，使贪婪变成非贪婪。

>>> s="This is a number 234-235-22-423"
>>> r=re.match(".+(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'4-235-22-423'
>>> r=re.match(".+?(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'234-235-22-423'
>>>

正则表达式模式中使用到通配字，那它在从左到右的顺序求值时，会尽量“抓取”满足匹配最长字符串，在上面的例子里面，“.+”会从字符串的启始处抓取满足模式的最长字符，其中包括想得到的第一个整型字段的中的大部分，“\d+”只需一位字符就可以匹配，所以它匹配了数字“4”，而“.+”则匹配了从字符串起始到这个第一位数字4之前的所有字符。

>>> re.match(r"aa(\d+)","aa2343ddd").group(1)
'2343'
>>> re.match(r"aa(\d+?)","aa2343ddd").group(1)
'2'
>>> re.match(r"aa(\d+)ddd","aa2343ddd").group(1) 
'2343'
>>> re.match(r"aa(\d+?)ddd","aa2343ddd").group(1)
'2343'
>>>

以上就是关于正则表达式基本知识，几乎所有的正则表达式都可以通过以上方法就行匹配，正则表达式广泛应用于检索，替换那些符合某个模式(规则)的文本，在python的爬虫应用中可以用来检索爬取检索到的有用信息。
