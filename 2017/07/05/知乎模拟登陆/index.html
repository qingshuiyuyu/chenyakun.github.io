知乎模拟登陆

直接上代码了，注意点都在注释！代码下面有重点说明！

#!/usr/bin/env python
# -*- coding:utf-8 -*-

import requests
import time
#引入Xpath解析器
from lxml import etree

#处理下载验证码
def captcha(cha):
    with open('captcha.jpg','w')as f:
        f.write(cha)
    return raw_input('请输入验证码：')

def zhihu_login():

    #1.重点：在登录过程中要保持cookie
    ss = requests.Session()
    #2.重点：此处url要与下面formdata的代码相对应
    url = 'https://www.zhihu.com/login/phone_num'#此处为手机账号登陆
    #url = 'https://www.zhihu.com/login/email'#此处为邮箱登陆

    #3.必须添加user-agent
    headers = {'User-Agent':'Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/60.0.3112.7 Safari/537.36'}
    #4.获取登录页面
    response = ss.get(url,headers=headers).content
    #转换成lxml
    html = etree.HTML(response)
    #5.获取_xsrf验证码
    _xsrf = html.xpath("//div[@class='view view-signin']//input[@name='_xsrf']/@value")

    #6.获取验证码链接，我直接写了，也可以跟url拼接
    captcha_url = "https://www.zhihu.com/captcha.gif?r=%d&type=login" % (time.time() * 1000)

    #7.获取验证码图片
    cha = ss.get(captcha_url,headers = headers).content

    #8.构建post
    formdata = {
        "_xsrf":_xsrf[0],#为了说明：我是用Xpath提取，_xsrf为一个元素的列表["值"]，可以写在上面
        "password":'xxxxxxx',#登陆密码
        "phone_num":'xxxxxxxxx',#你的注册手机账号--必须与url：(/login/phone_num)相对应
        #“email”:"xxx@163.com"#你的注册邮箱--必须与url：(/login/email)相对应
        'captcha':captcha(cha),#处理验证码
    }

    #9.把formdata里面的信息post到服务器登陆
    ss.post(url,headers=headers,data=formdata)
    #10.获取一个需要登陆后才能看到的页面，验证是否登陆成功
    response = ss.get('https://www.zhihu.com/settings/profile',headers=headers).content

    with open("shezhi.html", "w") as f:
            f.write(response)

if __name__=="__main__":
    zhihu_login()

重点：

1.登陆页面URL与在网页上看到的不一样，且要与post里面的账号类型相对应，可以通过抓包(推荐：Fiddler)获得

2.一定要带cookie和_cxrf发出请求,跳过网站防跨站攻击验证(cxrf)

3.验证码为手动输入，运行到输入验证码出，刷新文件夹，打开图片输入即可，后期可以加机器识别

4.注意formdata里面不要加”captcha_type”：”cn”,否则，验证码为汉字！
